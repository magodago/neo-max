# Proyecto practico 1

## Objetivos
- Desarrollar habilidades prácticas en la implementación de algoritmos de aprendizaje automático.
- Aprender a utilizar bibliotecas de Python para el análisis de datos y modelado.

## Contenido
En esta lección, se llevará a cabo un proyecto práctico que consiste en predecir el precio de las casas basándose en características como el tamaño del terreno, número de habitaciones y ubicación. Se utilizarán técnicas de aprendizaje supervisado para construir un modelo predictivo utilizando la biblioteca scikit-learn de Python. Los estudiantes aprenderán cómo importar datos, prepararlos para el análisis, seleccionar algoritmos adecuados y ajustar parámetros para mejorar la precisión del modelo.

## Ejercicio
1. Importa el conjunto de datos "housing.csv" que contiene información sobre casas.
2. Preprocesa los datos: maneja valores faltantes, normaliza características y divide el conjunto en datos de entrenamiento y prueba.
3. Implementa un modelo de regresión lineal utilizando scikit-learn y ajusta el modelo con los datos de entrenamiento.
4. Evalúa la precisión del modelo usando métricas como el error cuadrático medio (MSE) y el coeficiente de determinación (R²).
5. Experimenta con diferentes algoritmos de regresión (por ejemplo, regresión polinómica o árboles de regresión) y compara sus resultados.

## Resumen
- Se implementó un modelo de aprendizaje automático para predecir el precio de casas.
- Se utilizó la biblioteca scikit-learn en Python para el análisis de datos y modelado.
- Se evaluaron diferentes algoritmos de regresión para mejorar la precisión del modelo.
