# Revision

## Objetivos
- Comprender y aplicar técnicas avanzadas de revisión de código.
- Identificar y corregir errores comunes en la revisión de código.

## Contenido
En esta lección, se profundizará en las técnicas avanzadas para realizar una revisión eficaz del código. Se abordará cómo utilizar herramientas de análisis estático de código como SonarQube o ESLint para detectar problemas de calidad y seguridad antes de que lleguen a producción. Además, se discutirá la importancia de la comunicación efectiva durante las revisiones de código, incluyendo el uso de comentarios constructivos y el respeto por los compañeros de equipo.

Se explicará cómo organizar una sesión de revisión de manera productiva, priorizando aspectos como la claridad del código, la adherencia a convenciones y estándares, y la eficiencia en el mantenimiento futuro. Se proporcionarán ejemplos prácticos de cómo identificar y corregir errores comunes durante una revisión de código.

## Ejercicio
Realiza una revisión de código utilizando una herramienta de análisis estático (como ESLint) en un pequeño proyecto que hayas desarrollado previamente. Identifica al menos 5 problemas detectados por la herramienta y propón soluciones para cada uno. Luego, escribe comentarios constructivos sobre el código del compañero, enfocándote en áreas de mejora y aspectos positivos.

## Resumen
- Se destacará la importancia del uso de herramientas de análisis estático en la revisión de código.
- Se resaltarán las habilidades comunicativas necesarias para una revisión efectiva.
- Se identificarán y corregirán errores comunes durante el proceso de revisión.
