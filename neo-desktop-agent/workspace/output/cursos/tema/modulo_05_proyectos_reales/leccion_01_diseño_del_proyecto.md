# Diseño del proyecto

## Objetivos
- Comprender los elementos fundamentales del diseño de proyectos.
- Identificar las etapas y técnicas esenciales para la planificación inicial de un proyecto.

## Contenido
El diseño del proyecto es una fase crucial que establece el marco para su éxito. En esta lección, se abordarán los pasos necesarios para definir claramente el propósito, objetivos y alcance del proyecto. Se destacará la importancia de realizar un análisis exhaustivo de requisitos y la creación de un plan detallado que incluya tareas, recursos y cronograma. Además, se explicarán técnicas como la matriz de stakeholder para asegurar el compromiso y la participación de todos los involucrados.

## Ejercicio
El estudiante deberá diseñar un proyecto ficticio relacionado con su campo de estudio o interés personal. Deberá identificar el propósito del proyecto, definir sus objetivos específicos, realizar un análisis de requisitos básicos, y crear una matriz de stakeholder que muestre las expectativas y roles de cada parte interesada.

## Resumen
- El diseño del proyecto establece los cimientos para su éxito.
- Es fundamental identificar claramente el propósito, objetivos y alcance.
- La planificación inicial debe incluir análisis de requisitos y creación de un plan detallado.
