# Mantenimiento

## Objetivos
- Comprender los procesos y técnicas fundamentales para el mantenimiento de proyectos.
- Identificar las principales fases del ciclo de vida del mantenimiento de proyectos.

## Contenido
El mantenimiento en proyectos es una etapa crucial que asegura la continuidad y eficiencia del proyecto a lo largo del tiempo. Este proceso implica no solo la corrección de errores y la mejora de la funcionalidad, sino también la optimización de recursos y la adaptación al cambio. Es importante distinguir entre el mantenimiento correctivo (corregir fallos que surgen), preventivo (prevenir problemas potenciales) y adaptativo (adaptar el proyecto a nuevas condiciones o requisitos).

Durante este ciclo de vida, se realizan diversas actividades como la revisión del código fuente, actualización de documentación, pruebas y evaluaciones periódicas. Estas prácticas permiten mantener un alto nivel de calidad en el proyecto y facilitan su evolución hacia nuevas versiones o adaptaciones.

## Ejercicio
El estudiante deberá seleccionar un proyecto ficticio que ha sido entregado al final del curso y realizar un plan de mantenimiento para este proyecto. El plan debe incluir las fases de correctivo, preventivo y adaptativo, así como una descripción detallada de las actividades a realizar en cada fase.

## Resumen
- El mantenimiento es fundamental para asegurar la continuidad del proyecto.
- Incluye tres tipos principales: correctivo, preventivo y adaptativo.
- Un plan de mantenimiento completo debe abarcar todas estas fases.
