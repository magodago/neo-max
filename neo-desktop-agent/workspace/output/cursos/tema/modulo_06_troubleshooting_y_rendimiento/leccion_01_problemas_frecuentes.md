# Problemas frecuentes

## Objetivos
- Identificar y diagnosticar problemas comunes de rendimiento en sistemas informáticos.
- Implementar soluciones básicas para mejorar el rendimiento de un sistema.

## Contenido
En esta lección se abordarán los problemas frecuentes que pueden afectar al rendimiento de un sistema informático. Se tratará cómo identificar estos problemas a través de indicadores como la velocidad de procesamiento, la capacidad de respuesta y el uso del disco duro. Además, se explicarán técnicas básicas para diagnosticar y resolver dichos problemas, incluyendo la optimización del software, la gestión eficiente de la memoria RAM y la limpieza del sistema de archivos.

## Ejercicio
El estudiante deberá realizar un análisis de rendimiento en su propio equipo informático utilizando herramientas como Task Manager (Windows) o Activity Monitor (Mac). Deberá identificar posibles problemas y aplicar soluciones básicas, como cerrar programas no necesarios o liberar espacio en el disco duro. Posteriormente, se recomienda que compare los resultados antes y después de la optimización.

## Resumen
- Identificación de problemas de rendimiento mediante indicadores clave.
- Aplicación de técnicas básicas para mejorar el desempeño del sistema.
- Importancia de mantener un equipo limpio y eficiente en términos de recursos.
