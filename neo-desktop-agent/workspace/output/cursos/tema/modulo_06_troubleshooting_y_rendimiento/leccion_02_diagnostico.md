# Diagnostico

## Objetivos
- Comprender los fundamentos del diagnóstico de sistemas informáticos.
- Identificar y resolver problemas básicos mediante técnicas de diagnóstico.

## Contenido
El diagnóstico es una habilidad crucial para cualquier profesional en sistemas informáticos. Implica la identificación y resolución de problemas a través de un proceso sistemático que incluye la observación, análisis y corrección. Para diagnosticar correctamente, se deben seguir pasos específicos: primero, se debe recopilar toda la información relevante sobre el problema; luego, se analiza esta información para formular una hipótesis acerca de la causa del problema; finalmente, se verifica la hipótesis mediante pruebas y se toman medidas correctivas. Es importante recordar que el diagnóstico no solo busca resolver el problema actual, sino también prevenir futuros problemas.

## Ejercicio
El estudiante deberá diagnosticar un problema de rendimiento en una máquina virtual (VM). La VM tiene un sistema operativo Windows 10 y se ha notado un descenso significativo en la velocidad de respuesta. El estudiante debe recopilar información relevante, como el uso del CPU, memoria RAM y disco duro; formular una hipótesis sobre posibles causas; realizar pruebas para verificar la hipótesis; y proponer soluciones correctivas.

## Resumen
- Recolección de información es fundamental en el proceso de diagnóstico.
- El análisis y verificación de las hipótesis son pasos cruciales para resolver problemas técnicos.
- La resolución del problema debe considerar tanto la corrección inmediata como la prevención futura.
